# Norynthe.Score Dashboard

Static GitHub Pages root for `reports.norynthe.com`.

Current published surface:

- `index.html`: public Norynthe.Score dashboard.
- `norynthe-design-language.css`: shared typography and layout alignment layer.
- `Norynthe_master.png`: Norynthe wordmark asset.
- `favicon.png`: browser and touch icon.
- `CNAME`: GitHub Pages domain config for `reports.norynthe.com`.
- `robots.txt` and `sitemap.xml`: search indexing support for the dashboard root.
